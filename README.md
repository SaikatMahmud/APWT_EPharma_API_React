Implemented Laravel API, JWT and ReactJs. Have done the customer part only. <br/>
Some features include:
- Customer login/Reg
- Edit profile, profile pic
- Search medicines
- Edit/delete cart
- Place orders
- Order details/review/rating/cancellation/return
- PDF invoice
- Expense history, order statistics
